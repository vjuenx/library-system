===============================================
    KÜTÜPHANE YÖNETİM SİSTEMİ v1.0
===============================================

Bu yazılım, kütüphane otomasyonu için geliştirilmiş bir masaüstü uygulamasıdır.

📋 SİSTEM GEREKSİNİMLERİ:
- Windows 10 veya üzeri
- .NET 9.0 Runtime (otomatik yüklenecek)
- En az 100 MB boş disk alanı

🚀 KURULUM TALİMATLARI:
1. "KURULUM.bat" dosyasını çift tıklayın
2. Kurulum tamamlandığında masaüstünde kısayol oluşacak
3. Uygulamayı başlatmak için kısayola çift tıklayın

👤 VARSAYILAN KULLANICILAR:
- admin / 123456 (Yönetici)
- librarian / 123456 (Kütüphaneci)
- user / 123456 (Kullanıcı)

📚 ÖZELLİKLER:
✅ Kullanıcı girişi ve rol yönetimi
✅ Görevli yönetimi (Admin)
✅ Öğrenci yönetimi
✅ Entity Framework ile veritabanı
✅ SQLite veritabanı (taşınabilir)

🔧 SORUN GİDERME:
- Uygulama açılmıyorsa .NET 9.0 Runtime yükleyin
- Veritabanı hatası alırsanız uygulamayı yönetici olarak çalıştırın
- Antivirus yazılımı uygulamayı engelleyebilir
